## 특수 요청 파라미터

- `language_preference`: ISO 639-1 언어 코드 (예: "en", "ko"). 응답 언어를 지정합니다.
- `max_steps`: 최대 연구 단계 수를 지정합니다. 최소 1이상 최대 10이하의 정수입니다.
- `models`: 모델 폴백 체인입니다.
- `preset`: 사전 설정된 모델 구성을 사용합니다. 예: "fast-research", "deep-research".
