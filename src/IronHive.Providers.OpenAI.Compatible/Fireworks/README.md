## 모델 명명 규칙

Fireworks는 특수한 모델 ID 형식을 사용합니다:
예: `accounts/{account_id}/models/{model_name}`
- `accounts/{account_id}`: Fireworks 계정 ID
- `models/{model_name}`: 모델 이름
